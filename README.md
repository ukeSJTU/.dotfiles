# Troubleshooting

创建 symlinks 的方法似乎会破坏 wezterm 这些实时更新的软件。或者说你在.dotfiles 下修改了文件，需要重新 symlink 一下，才能让 wezterm 读取到配置。具体讨论和解决办法请看https://github.com/wezterm/wezterm/issues/1697
